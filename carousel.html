About the code

While building a slide show is a fairly simple task thanks to libraries such as jQuery and Bootstrap, I felt that, as a new developer, it would be a challenging learning experience to build one from scratch using only Javascript (okay, and jQuery when I'm stumped)

All the images below come from a loop that pulls information from Javascript objects which contain the necessary information (src, alt, id etc.) When an image is clicked, a function is called which pulls the information from the appropriate object, and then uses AJAX to insert the elements into the code.
